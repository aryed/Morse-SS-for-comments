A short discussion about the framework of the paper: the main two "models" for flow categories are \cite{abouzaid2024foundation} and \cite{CJS95}. Note that the first one is the only framework that construct morphisms between flow categories, and their cofibers, hence we can use it directly to define a spectral sequence of a filtered object in $\infty$ category. The main disadvantage of this framework is the lack of simple description to the index filtration, given a structure flow category $\fC$, the map $\fC_{[p,-\infty]} \to \fC_{[p-1,-\infty]}$ involving the diagonal bimodule, which is far from being simple, hence it's cone and the intire spectral sequence. On the other hand the CJS-like framework in the literature, for example \cite{cote2023equivariant,bonciocat2024revisiting}, lack the full categorical structure. In \cite{cote2023equivariant}, a flow bimodule $W:\fC \to \fC'$ induces a map on the realization $\norm{\fC} \to \norm{\fC'}$, but lack the description of a composition of flow bimodules, in particular don't construct a functor to $\Sp$.

Given a flow category $\fC$, one can use the index function of $\fC$ to define a filtered flow category $\fC_p$, where the maps $\fC_p \to \fC_{p+1}$ are given by diagonal bimodules. Then to realize it to a filtered spectrum and produce a spectral sequence out of it.
